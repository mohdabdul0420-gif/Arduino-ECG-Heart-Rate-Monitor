# Arduino-Based ECG Monitoring System with Bluetooth

## Overview

This project implements a portable ECG monitoring system using Arduino, an AD8232 ECG sensor, and an HC-05 Bluetooth module. The system captures heart electrical signals and transmits them wirelessly to a smartphone or computer for real-time monitoring.

## Features

* Real-time ECG signal acquisition
* Bluetooth-based wireless communication
* Portable and low-cost design
* Easy integration with mobile and desktop applications

## Components Used

* Arduino Uno/Nano
* AD8232 ECG Sensor
* HC-05 Bluetooth Module
* ECG Electrodes
* Breadboard and Jumper Wires

## Working

The ECG sensor captures heart activity through electrodes attached to the body. Arduino reads the sensor output and transmits the processed data through Bluetooth. The received data can be visualized using a Bluetooth terminal or serial plotting application.

## Applications

* Remote patient monitoring
* Telemedicine
* Biomedical research
* Home healthcare systems
